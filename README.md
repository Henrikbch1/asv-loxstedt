# asv-loxstedt
Neuentwicklung der Vereinsplattform für den ASV Loxstedt e.V. | Migration von Strato-Baukasten zu einer maßgeschneiderten Fullstack-Lösung (Spring Boot &amp; React TS) für maximale Performance, optimierte UX und volle Individualisierbarkeit
# 🎣 ASV Loxstedt - Von der Standardlösung zum High-Performance Portal

Dieses Projekt markiert den technologischen Wendepunkt für den **ASV Loxstedt e.V.**. Die bisherige Lösung auf Basis eines Strato-Baukastens stieß an ihre Grenzen in Bezug auf Design, Performance und funktionale Erweiterbarkeit.

**Die Mission:** Eine von Grund auf neu entwickelte Plattform, die exakt auf die Bedürfnisse der Vereinsmitglieder und des Vorstands zugeschnitten ist.

---

## 🚀 Warum dieser Technologiewechsel?

* **Performance:** Blitzschnelle Ladezeiten durch eine entkoppelte Architektur (React Frontend + Spring Boot API) im Vergleich zum trägen Baukasten-System.
* **UX & UI:** Maßgeschneidertes Design statt vorgefertigter Templates – für eine intuitive Bedienung, egal ob am PC oder mobil am Vereinsgewässer.
* **Full Individualization:** Entwicklung spezifischer Features (z.B. digitale Fangmeldungen, Mitgliederbereich), die mit Standard-Systemen nicht umsetzbar waren.
* **Scale & Security:** Typsichere Entwicklung mit **TypeScript** und ein robustes **Java-Backend** für langfristige Wartbarkeit.
